## Synopsis

My own printf function

Available option :

-> width
-> precision

Flags :

-> '#'
-> '0'
-> '-'
-> '+'
-> ' '

Formats :

-> %%
-> %s
-> %S
-> %c
-> %C
-> %i
-> %d
-> %D
-> %x
-> %X
-> %p
-> %o
-> %O
-> %u
-> %U

## Installation

type 'Make' in the root directory of the project
compile the created library with your program : 'cc libftprintf.a exemple.c'
run the created binary : './a.out'
